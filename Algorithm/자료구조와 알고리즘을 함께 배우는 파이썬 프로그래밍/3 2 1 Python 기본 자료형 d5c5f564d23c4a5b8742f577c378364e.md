# 3.2.1 Python 기본 자료형

복습: No
블로그: No

| 자료형 |  | 변경가능
(mutable) | 반복가능
(iterable) | remarks |
| --- | --- | --- | --- | --- |
| boolean | bool |  |  |  |
| numeric | int |  |  |  |
|  | float |  |  |  |
|  | complex |  |  |  |

|  | bytes |  |  |  |

|  | memoryview |  |  |  |
|  | list |  |  |  |
|  | tuple |  |  |  |
|  | range |  |  |  |
| mapping | dict |  |  |  |
| set | set |  |  |  |
|  | frozenset |  |  |  |

객체의 고유번호 (object identifier) 변경 없이 값이 변경될 수 있으면 변경 가능 (mutable)

고유번호 변경 없이 값을 변경될 수 없으면 변경 불가능 (immutable)

자료형이 반복가능 (iterable)이라는 표현은 반복문 for에서 자료형 객체에 포함된 원소들은 한 번에 하나씩 꺼내어 사용할 수 있는 자료형을 의미한다.
