# 2.2.3 포맷 지정 데이터 출력

## 문자열 서식 지정

### %d, %s, %f의 형식 지정자를 사용한 출력 형식 지정

```python
print("%d %5d %10s, %7.3f" % (123, 456, "Python", 123.456))
```

|형식|설명|
|---|-------------|
|%d|10진수를 출력|
|%5d|5칸을 확보하여 오른쪽으로 정렬된 10진수로 출력|
|%05d|5칸을 확보하여 오른쪽으로 정렬된 10진수로 출력하며, 왼쪽에 남은 공간에 9을 출력|
|%c|문자 출력|
|%s|문자열(string) 출력|
|%10s|10칸을 확보하여 오른쪽으로 정렬된 문자열을 출력|
|%-10s|10칸을 확보하여 왼쪽으로 정렬된 문자열을 출력|
|%f|실수 출력|
|%7.3f|7칸을 확보하여 오른쪽으로 정렬된 실수를 출력하며, 소수점 아래 3자리까지 출력하며, 오른쪽 빈칸은 0으로 채움|

### {n:x}와 .format 메소드를 사용한 서식 지정

```python
print("{0:d} {1:5d} {2:05d} {3:10s} {4:7.3f}".format(123, 456, 678, "Python", 123.456))
```

|형식|설명|
|----|----------|
|{i:>}|i번째 항을 오른쪽 정렬|
|{i:^}|중앙정렬|
|{i:<}|왼쪽 정렬|

### format() 함수를 사용한 서식 지정

```python
format(123456789, '.d')
# 1000 단위로 comma(,)를 삽입한 형식으로 출력
```


### e.g.

```python
# Python print() width %formatting
print("%5d %7d %07d %10s %7.3f" % (123, 456, 678, "Python", 123.456789))
print("%5d %7d %07d %-10s %10.6f" % (12, 34, 5, "Python", 123.456789))

# Python print() with .format()
print("{0:5d} {1:7d} {2:07d} {3:10s} {4:7.3f}".format(123, 456, 678, "Python", 123.456789))
print("{0:5d} {1:7d} {2:07d} {3:<10s} {4:10.6f}".format(12, 34, 5, "Python", 123.456789))
print("{0:5d} {1:7d} {2:07d} {3:^10s} {4:10.6f}".format(12, 34, 5, "Python", 123.4567))
print("{0:5d} {1:7d} {2:07d} {3:>10s} {4:10.6f}".format(12, 34, 5, "Python", 123.4567))
print("Long integer with comma at each 1000 unit : ", format(123456789, ',d'))
#--==>>
'''
  123     456 0000678     Python 123.457
   12      34 0000005 Python     123.456789
  123     456 0000678 Python     123.457
   12      34 0000005 Python     123.456789
   12      34 0000005   Python   123.456700
   12      34 0000005     Python 123.456700
Long integer with comma at each 1000 unit :  123,456,789
'''
```
