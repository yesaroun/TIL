
# 1. **평균값 계산기**

파라미터로 정수 배열 **`intArray`** 를 받고 배열의 모든 값의 평균을 리턴해주는 메소드 **`computeAverage`** 를 쓰세요. 리턴값의 자료형은 **`double`** 입니다.
```
4.3333333
3.75
4.0
3.0
```

## 템플릿
AverageFinder.java
```python
public class AverageFinder {
    double computeAverage(int[] intArray) {
        // 코드를 입력하세요.
    }
}
```
Main.java
```python
public class Main {
    public static void main(String[] args) {
        AverageFinder finder = new AverageFinder();

        // 테스트 1
        int[] testArray1 = {3, 7, 3};
        System.out.println(finder.computeAverage(testArray1));

        // 테스트 2
        int[] testArray2 = {1, 3, 4, 7};
        System.out.println(finder.computeAverage(testArray2));

        // 테스트 3
        int[] testArray3 = {4};
        System.out.println(finder.computeAverage(testArray3));

        // 테스트 4
        int[] testArray4 = {1, 2, 3, 4, 5};
        System.out.println(finder.computeAverage(testArray4));
    }
}
```

## 해설
### **합 구하기**
일단 반복문을 이용해서 배열의 모든 값을 더해야 한다.

### **while문 이용**
```java
double computeAverage(int[] intArray) {
    int sum = 0;
    int i = 0;

    while (i < intArray.length) {
        sum += intArray[i];
        i++;
    }
}
```

### **기본 for문 이용**
```java
double computeAverage(int[] intArray) {
    int sum = 0;
    for (int i = 0; i < intArray.length; i++) {
        sum += intArray[i];
    }
}
```

### **for each 이용**
```java
double computeAverage(int[] intArray) {
    int sum = 0;
    for (int num : intArray) {
        sum += num;
    }
}
```

### **평균 구하기**
```java
return (double) sum / intArray.length;
```

## **모범 답안**
```java
public class AverageFinder {
    double computeAverage(int[] intArray) {
        int sum = 0;
        for (int num : intArray) {
            sum += num;
        }

        return (double) sum / intArray.length;
    }

    public static void main(String[] args) {
        AverageFinder finder = new AverageFinder();

        // 테스트 1
        int[] testArray1 = {3, 7, 3};
        System.out.println(finder.computeAverage(testArray1));

        // 테스트 2
        int[] testArray2 = {1, 3, 4, 7};
        System.out.println(finder.computeAverage(testArray2));

        // 테스트 3
        int[] testArray3 = {4};
        System.out.println(finder.computeAverage(testArray3));

        // 테스트 4
        int[] testArray4 = {1, 2, 3, 4, 5};
        System.out.println(finder.computeAverage(testArray4));
    }
}
//--==>>
/*
4.333333333333333
3.75
4.0
3.0
*/
```


# 2. **두 원소의 차 중 최댓값 구하기**

파라미터로 정수 배열 **`intArray`** 를 받고, 두 원소의 차이 중 가장 큰 값을 리턴해주는 메소드 **`greatestDifference`** 를 쓰세요. 리턴값의 자료형은 **`int`** 입니다. 만약 원소 개수가 2개보다 작으면 **`0`** 을 리턴해줍니다.
```
9
13
3
0
8
0
```

## 해설
원소 개수가 2개보다 작으면 **`0`** 을  리턴
```java
// 원소가 0개 또는 1개면 0 리턴
if (intArray.length < 2) {
    return 0;
}
```

두 원소의 차이 중 가장 큰 값을 구하려면, 배열의 최댓값 빼기 최솟값을 하면 되겠죠? 먼저 최댓값과 최솟값을 구하려면 이렇게 하면 되고:
```java
// 최댓값, 최솟값 구하기
int max = intArray[0];
int min = intArray[0];

for (int num : intArray) {
    if (num > max) {
        max = num;
    }

    if (num < min) {
        min = num;
    }
}
```

그 차이를 리턴해주면 끝납니다.
```java
// 최대 차 리턴
return max - min;
```

### **모범 답안**
**GreatestDifferenceFinder.java**
```java
public class GreatestDifferenceFinder {
    int greatestDifference(int[] intArray) {
        if (intArray.length < 2) {
            return 0;
        }

        int max = intArray[0];
        int min = intArray[0];

        for (int i = 0; i < intArray.length; i++) {
            if (max < intArray[i]) {
                max = intArray[i];
            }

            if (min > intArray[i]) {
                min = intArray[i];
            }
        }
        return max - min;
    }
}
//--==>>
/*
9
13
3
0
8
0
*/
```
