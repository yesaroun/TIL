# 장고로 만든 웹 사이트의 작동 구조

1. 클라이언트가 url 접속 
2. [urls.py](http://urls.py) 확인
3. [views.py](http://views.py) 확인
4. [models.py](http://models.py) 확인
5. 데이터베이스에서 필요한 자료를 받거나 저장
6. 4, 3번으로 돌아가고
7. 준비한 자료(DB에서 가져온 자료)를 템프릿에 채워 출력

# MTV 패턴이란?

장고로 만든 웹 사이트는 모델(model)로 자료의 형태를 정의하고, 뷰(view)로 어떤 자료를 어떤 동작으로 보여줄 지 정의하고, 템플릿(template)으로 웹 페이지에서 출력할 모습을 정의한다.
이러한 작동 구조를 MTV 패턴이라고 부른다.
